# Melon-Chart-Analysis

학교 수업 데이터 과학을 한 학기 동안 수강하며 과제로 작성한 코드입니다.  
파이썬을 활용한 데이터 수집 -> 전처리 -> 시각화 -> 간단한 NLP(감정 분석)까지의 과정을 단계적으로 정리한 개인 학습 기록용 레포지토리입니다.

---

## 사용 도구 및 환경

언어: Python
환경: Jupyter Notebook
주요 라이브러리: pandas, matplotlib/seaborn, scikit-learn, konlpy, wordcloud 등

---

## 내용 요약

### 01 – 데이터 수집 및 통합
- 차트 데이터 수집
- 여러 소스 데이터 병합 및 컬럼 정규화

### 02 – 데이터 전처리
- 결측치 처리, 중복 제거
- 텍스트 정제(특수문자 제거, 토큰화 등)

### 03 – 탐색적 데이터 분석(EDA)
- 빈도 분석
- 상위 키워드 추출

### 04 – 시각화
- 단어 빈도 시각화(막대그래프, 워드클라우드)
- 분포 및 추이 시각화

### 05 – 감정 분석(NLP 기초)
- 가사/텍스트 기반 감정 분류
- 결과 요약 및 비교 분석
